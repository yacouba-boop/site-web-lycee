# 🏫 Site Web du Lycée – Projet Front-End

Ce projet consiste en la création d’un site web moderne destiné à un lycée, afin de faciliter l’accès aux informations scolaires pour les élèves, enseignants et visiteurs.  
Il s’agit d’un travail entièrement orienté **front-end**, conçu avec HTML et CSS, dans un style simple, propre et responsive.

---

![Aperçu du site](images/fullpage.png)

---

## 🛠️ Technologies utilisées

- **HTML5**
- **CSS3**
- **Responsive Design**
- **Images WebP** pour optimiser les performances

---

## ✨ Fonctionnalités principales

- Page d’accueil présentant le lycée  
- Navigation simple et intuitive  
- Design moderne adapté aux jeunes utilisateurs  
- Mise en page responsive pour ordinateurs et mobiles  
- Utilisation d’images optimisées pour réduire le temps de chargement
